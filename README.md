# ts-react-snippets

React typescript simple snippets for vs code.

## Snippets


| Snippet                      | Purpose                                                      |
| ---------------------------- | ------------------------------------------------------------ |
| `tsc`                        | React component                                              |
| `tspc`                       | React pure component                                         |
| `tsfc`                       | React functional component                                   |
| `tsm`                        | React memoized functional component                          |
| `react`                      | import react                                                 |
